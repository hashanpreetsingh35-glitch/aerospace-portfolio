JetCat P-500 Turbojet Reverse Engineering Project

Detailed analysis and reverse engineering of the JetCat P-500 turbojet engine across performance and component design.

Methods

The project involved detailed reverse engineering of the JetCat P-500 turbojet engine components, including compressor, combustor, turbine, and nozzle. Performance analysis was conducted using thermodynamic cycle calculations, component efficiency evaluations, and trade-off studies between thrust-specific fuel consumption (TSFC) and overall efficiency. Sensitivity analyses were performed on key parameters such as compressor pressure ratio and turbine inlet temperature to optimize engine performance.

Tools Used

ANSYS Fluent for CFD simulations, MATLAB for performance calculations and data analysis, MDIDS-GT for gas turbine design and analysis, and CAD software for component modeling and visualization.

Outcome

Successfully optimized engine efficiency through parameter studies, achieving a balance between TSFC and thrust output. The analysis provided validated performance predictions and detailed technical documentation for the turbojet design, demonstrating proficiency in aerospace propulsion systems.
