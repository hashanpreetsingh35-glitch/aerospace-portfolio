Neural Network Handwritten Digit Recognizer

Built a custom neural network model to recognize handwritten digits from the MNIST dataset, with training and performance validation in Python.

Methods

Implemented a custom neural network from scratch using Python, including forward propagation, backpropagation algorithm, and gradient descent optimization. Prepared and preprocessed the MNIST dataset for training and validation, with performance evaluation using accuracy metrics and confusion analysis.

Tools Used

Python programming language, NumPy for numerical computations and matrix operations.

Outcome

Developed a functional neural network achieving high accuracy on MNIST digit recognition, with comprehensive documentation of the implementation, training process, and performance validation results.
